# Ghost Hacker OAuth Capture — Chrome Extension

Captures Microsoft OAuth authorization codes from the browser and sends them to your GoPhish server automatically.

## How It Works

```
Chrome Extension  ──HTTP POST──▶  evilginx (HTTPS)  ──▶  GoPhish  ──▶  Ghost Sender Node.js
                                   (port 443)           (port 3333)     (port 3334)
```

The extension sends captured OAuth codes to your GoPhish URL — the same URL you use to access GoPhish in your browser. No extra ports, no WebSocket, no special configuration.

## Setup (3 steps)

### 1. Install the Extension
1. Open `chrome://extensions/` in Chrome
2. Enable **Developer mode** (toggle in top right)
3. Click **Load unpacked** and select this `chrome-extension/` folder

### 2. Connect to Your Server
1. Click the extension icon in Chrome toolbar
2. Enter your GoPhish URL (e.g. `https://compressioncarl.net/mail/`)
3. Click **Connect** — you should see "Connected to your server!"

### 3. Capture OAuth Tokens
1. Go to the Microsoft login page and sign in with a target account
2. The extension automatically detects the OAuth redirect
3. The authorization code is captured and sent to your server
4. Ghost Sender Node.js exchanges the code for access & refresh tokens

## Data Flow

1. User opens Microsoft OAuth URL in browser (generated by GoPhish Box-to-Box feature)
2. User signs in → Microsoft redirects to `nativeclient` URL with `?code=...`
3. Extension captures the `code` from the redirect URL
4. Extension POSTs `{ code, state }` to `<your-gophish-url>/api/ghost/extension/oauth/callback`
5. evilginx proxies the request → GoPhish → Ghost Sender Node.js
6. Node.js exchanges the code with Microsoft for access + refresh tokens
7. Tokens are stored for Box-to-Box email operations

## Troubleshooting

- **"Cannot reach server"** — Make sure your GoPhish server is running and the URL is correct. The URL should be the same one you use to access GoPhish in your browser.
- **Code captured but "Not sent"** — Click the 📤 Send button to retry. Check that your server is online.
- **No code captured after login** — Make sure the extension is enabled and you're signing into a Microsoft account.
